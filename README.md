
# Digital Café N°1 

<ul>
	<li>importer le projet</li>
	<li>faire un mvn clean </li>
	<li> importer la base de données se trouvant dans le dossier "ressource"</li>
	<li>insérer les valeurs du login de serveur mysql dans le fichier pom.xml du projet </li> 
	<li>update le projet Maven </li>
	<li>taper la commande mvn package </li>
	<li>taper la commande mvn tomcat7:run-war </li>
	<li>saisir l'url suivant sur le navigateur: localhost:9000/digital-cafe-001-groupe3</li>
</ul>


<h2>Pour déployer le projet </h2>

<p>pour le déployer le projet, il faut insérer les informations de login du serveur Tomcat de votre machine qui a les droits de manager, manager-script</p>
<p>taper la commande mvn package </p>
<p>taper la commande mvn tomcat7:deploy</p>
